Este é o primeiro post do blog! 🎉

Para escrever um novo post, basta criar um novo arquivo dentro da pasta
`_posts/` seguindo o padrão de nome:

```
AAAA-MM-DD-titulo-do-post.md
```

Por exemplo: `2026-07-20-minhas-ferias.md`.

No topo do arquivo (entre os `---`), coloque o título e a data. Depois é só
escrever o conteúdo em Markdown normalmente (títulos com `#`, *negrito* com
`**texto**`, listas, links, etc).
